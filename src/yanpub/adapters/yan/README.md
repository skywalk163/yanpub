# 言 (Yan) 适配器

> 前缀调用风格的中文编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `yan` |
| 版本 | 3.0.0 |
| 文件扩展名 | `.言`、`.yan` |
| 注释语法 | `--` |
| REPL 提示符 | `言> ` |

## 安装

言基于 Python 实现，与墨言共享同一后端项目（atomyan）：

```bash
# 确保已安装 Python 3.8+
# 言项目位于本地
python yan.py --help
```

运行、求值和交互模式：

```bash
python yan.py <file>          # 运行文件
python yan.py -e '代码'       # 求值表达式
python yan.py -i              # 进入 REPL
```

## 语法示例

### Hello World

```yan
打印("你好，世界！")

定义 甲 = 42
定义 乙 = 相乘 甲 2
打印(乙)

定义 求和 = 函数 甲 乙 那么 相加 甲 乙

定义 结果 = 求和 3 5
打印(结果)
```

### 斐波那契数列

```yan
定义 斐波那契 = 函数 甲 那么 如果 小于 甲 2 那么 甲 否则 相加 (斐波那契 (相减 甲 1)) (斐波那契 (相减 甲 2))

定义 甲 = 0
每当 小于 甲 10 时候：
  打印 格式化 "斐波那契({}) = {}" 甲 (斐波那契 甲)
  赋值 甲 = 相加 甲 1
。
```

## 已知限制

- 前缀调用风格对初学者有一定学习曲线
- 不支持调试器（debug）功能
- 注释语法为 `--`，与部分中文编程语言常用的 `#` 不同
