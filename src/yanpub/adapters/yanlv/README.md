# 言律 (YanLv) 适配器

> 意合式中文编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `yanlv` |
| 版本 | 2.0.0 |
| 文件扩展名 | `.律`、`.yan` |
| 注释语法 | `#` |
| REPL 提示符 | `言律> ` |

## 安装

言律基于 Python 实现，依赖 jieba 中文分词库：

```bash
# 确保已安装 Python 3.8+
pip install jieba

# 运行文件
python -m yanlv 运行 <file>

# 进入 REPL
python -m yanlv 交互
```

## 语法示例

### Hello World

```yanlv
输出("你好，世界！")

定 甲 = 42
定 乙 = 甲 * 2
输出(乙)

函数 求和(甲, 乙) {
  返回 甲 + 乙
}

定 结果 = 求和(3, 5)
输出(结果)
```

### 斐波那契数列

```yanlv
函数 斐波那契(甲) {
  如果 甲 < 2 那么 {
    返回 甲
  }
  返回 斐波那契(甲 - 1) + 斐波那契(甲 - 2)
}

定 甲 = 0
当 甲 < 10 {
  输出("斐波那契(" + 甲 + ") = " + 斐波那契(甲))
  甲 = 甲 + 1
}
```

## 已知限制

- 无 eval 命令，代码片段执行依赖临时文件回退机制
- 依赖 jieba 分词库，启动时需加载词典
- 意合式语法（因果链、语境省略）需要一定学习适应
- 不支持包管理器和 WASM 执行
