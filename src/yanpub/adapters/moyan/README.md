# 墨言 (Moyan) 适配器

> 基于字节码虚拟机的中文编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `moyan` |
| 版本 | 3.0.0 |
| 文件扩展名 | `.墨`、`.moyan` |
| 注释语法 | `--` |
| REPL 提示符 | `墨言> ` |

## 安装

墨言与言共享同一后端项目（atomyan），使用 `--vm` 选项启用虚拟机后端：

```bash
# 确保已安装 Python 3.8+
python yan.py --vm <file>       # 运行文件（VM 模式）
python yan.py --vm -e '代码'    # 求值表达式（VM 模式）
python yan.py --vm -i           # 进入 REPL（VM 模式）
```

## 语法示例

### Hello World

```moyan
打印("你好，世界！")

定义 甲 = 42
定义 乙 = 甲 * 2
打印(乙)

函数 求和(甲, 乙) {
  返回 甲 + 乙
}

定义 结果 = 求和(3, 5)
打印(结果)
```

### 斐波那契数列

```moyan
函数 斐波那契(甲) {
  如果 甲 < 2 那么 {
    返回 甲
  }
  返回 斐波那契(甲 - 1) + 斐波那契(甲 - 2)
}

定义 甲 = 0
当 甲 < 10 时候 {
  打印("斐波那契(" + 甲 + ") = " + 斐波那契(甲))
  甲 = 甲 + 1
}
```

## 已知限制

- 与言共享同一后端项目，需通过 `--vm` 选项区分运行模式
- 注释语法为 `--`，与部分中文编程语言常用的 `#` 不同
- 不支持调试器（debug）功能
