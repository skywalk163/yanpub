# 段言 (Duan) 适配器

> 用段落书写的编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `duan` |
| 版本 | 1.3.8 |
| 文件扩展名 | `.段`、`.duan` |
| 注释语法 | `#` |
| REPL 提示符 | `段言> ` |

## 安装

段言基于 Python 实现，需要安装段言运行时：

```bash
# 确保已安装 Python 3.8+
# 段言项目位于本地，无需 pip 安装
python cli/duan.py --help
```

运行文件时使用 `--backend src` 选项以避免 ANTLR ATN 版本不匹配问题：

```bash
python cli/duan.py run --backend src <file>
```

## 语法示例

### Hello World

```duan
打印("你好，世界！")。

设甲为四十二。
设乙为甲乘二。
打印(乙)。

段落 求和 参数 甲 乙
  返回 甲 加 乙。
结束

设结果为 求和(三, 五)。
打印(结果)。
```

### 变量与运算

```duan
打印("你好，世界！")。

设甲为四十二。
设乙为甲乘二。
打印("四十二乘二等于：")。
打印(乙)。

段落 求和 参数 甲 乙
  返回 甲 加 乙。
结束

设结果为 求和(三, 五)。
打印("三加五等于：")。
打印(结果)。
```

### 斐波那契数列

```duan
段落 斐波那契 参数 甲
  如果 甲 小于 二 那么：
    返回 甲。
  结束
  返回 斐波那契(甲 减 一) 加 斐波那契(甲 减 二)。
结束

设甲为零。
当 甲 小于 十：
  打印("斐波那契(" + 转字符串(甲) + ") = " + 转字符串(斐波那契(甲)))。
  设甲为 甲 加 一。
结束
```

## 已知限制

- 无内置 eval 命令，代码片段执行依赖临时文件回退机制
- 无独立 REPL 模式，需通过 Playground 或 LSP 进行交互
- ANTLR 后端存在 ATN 版本兼容问题，运行时需指定 `--backend src`
- 不支持 WASM 执行和包管理器
