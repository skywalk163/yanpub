# 知行 (Zhixing) 适配器

> 管道式中文编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `zhixing` |
| 版本 | 0.9.0 |
| 文件扩展名 | `.行`、`.yan` |
| 注释语法 | `#` |
| REPL 提示符 | `知行> ` |

## 安装

知行基于 Python 实现，通过子进程调用后端：

```bash
# 确保已安装 Python 3.8+
python -m yan.cli <file>       # 运行文件
python -m yan.cli -c '代码'    # 求值表达式
python -m yan.cli -i           # 进入 REPL
```

## 语法示例

### Hello World

```zhixing
"你好，世界！"，印。

定甲是42。
定乙是甲乘2。
乙，印。

10加5，印。
3乘4加2，印。

5大3，印。
真且真，印。
若5大3则"大"否则"小"，印。
```

### 变量与运算

```zhixing
"你好，世界！"，印。

定甲是42。
定乙是甲乘2。
乙，印。

10加5，印。

# 阶乘函数
定fact是函n：
  若n等1则1否则n乘fact $(n-1)。
。

fact 5，印。

# 列表管道
范围10，皆乘2，印。
范围10，只大5，归加0，印。
```

### 斐波那契数列

```zhixing
定fib是函n：
  若n小等于2则1否则fib $(n-1)加fib $(n-2)。
。

fib 1，印。
fib 2，印。
fib 6，印。
fib 10，印。
```

## 关键字

定、设、函、若、则、否则、当、遍历、入、于、真、假、空、尝试、捕获、结束、完毕、导入、导出、模块

## 已知限制

- 不支持包管理器和 WASM 执行
- 管道式语法风格与常见编程语言差异较大
- 不支持调试器（debug）功能
