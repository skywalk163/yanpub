# 翰语 (Hanyu) 适配器

> 基于 LLVM 的中文编程语言，使用百家姓标识符

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `hanyu` |
| 版本 | 0.6.0 |
| 文件扩展名 | `.翰`、`.hanyu` |
| 注释语法 | `#` |
| REPL 提示符 | `翰> ` |

## 安装

翰语基于 LLVM IR 进行代码生成，支持 JIT 和子进程两种执行模式：

```bash
# 确保已安装 Python 3.10+
# 运行文件
python -m hanyu.compiler <file.翰>

# 交互式 REPL
python -m hanyu.repl
```

## 语法示例

### Hello World

```hanyu
打印("你好世界")

定义赵x=42
定义赵y=赵x乘2
打印(赵y)

函数赵求和(赵a, 赵b):
  返回赵a加赵b

定义赵结果=赵求和(3, 5)
打印(赵结果)
```

### 百家姓标识符

翰语使用百家姓作为变量前缀，实现命名空间式的作用域隔离：

```hanyu
定义赵x=42    # 赵姓变量
定义孙lst=[3 5 8]  # 孙姓变量
定义吴name="张三"  # 吴姓变量
```

### 斐波那契数列

```hanyu
函数赵fib(赵n):
  如果赵n小于2那么:
    返回赵n
  返回赵fib(赵n减1)加赵fib(赵n减2)

定义赵i=0
循环当满赵i小于10:
  打印(赵fib(赵i))
  赵i=赵i加1
```

## 已知限制

- 编译型语言，无单行 eval 模式
- LLVM 后端依赖较重，构建环境要求较高
- 不支持包管理器和调试器
- 部分 Windows 环境下 GC 运行时需 pthread 兼容层
