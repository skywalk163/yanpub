# 翰语 (Hanyu) 适配器

> 基于 LLVM 的中文编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `hanyu` |
| 版本 | 0.5.0 |
| 文件扩展名 | `.翰`、`.hanyu` |
| 注释语法 | `#` |
| REPL 提示符 | 无（不支持 REPL） |

## 安装

翰语基于 LLVM IR 进行代码生成，支持 JIT 和子进程两种执行模式：

```bash
# 确保已安装 Python 3.8+
# 运行文件
python -m hanyu.compiler <file>
```

翰语无内置 REPL，但可通过 Playground HTTP 服务进行交互式编程。

## 语法示例

### Hello World

```hanyu
打印(42)

定义 甲 = 42
定义 乙 = 甲 * 2
打印(乙)

函数 求和(甲, 乙) {
  返回 甲 + 乙
}

定义 结果 = 求和(3, 5)
打印(结果)
```

### 斐波那契数列

```hanyu
函数 斐波那契(甲) {
  如果 甲 < 2 那么 {
    返回 甲
  }
  返回 斐波那契(甲 - 1) + 斐波那契(甲 - 2)
}

定义 甲 = 0
当满 甲 < 10 {
  打印("斐波那契(" + 甲 + ") = " + 斐波那契(甲))
  甲 = 甲 + 1
}
```

## 已知限制

- 无 REPL 支持，交互式编程需通过 Playground HTTP 服务
- 无 eval 命令，代码片段执行依赖临时文件回退
- LLVM 后端依赖较重，构建环境要求较高
- 不支持包管理器和调试器
