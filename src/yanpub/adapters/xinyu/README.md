# 心语 (XinYu) 适配器

> 安全、简洁的中文编程语言

## 基本信息

| 属性 | 值 |
|------|-----|
| 语言 ID | `xinyu` |
| 版本 | 0.9.0 |
| 文件扩展名 | `.心`、`.xinyu` |
| 注释语法 | `#` |
| REPL 提示符 | `心语> ` |

## 安装

心语基于 Python 实现，通过子进程调用后端：

```bash
# 确保已安装 Python 3.8+
python src/main.py <file>       # 运行文件
python src/main.py -c '代码'    # 求值表达式
python src/main.py              # 进入 REPL
```

## 语法示例

### Hello World

```xinyu
打印("你好，世界！")

定 甲 = 42
定 乙 = 甲 * 2
打印(乙)

函数 求和(甲, 乙) {
  返回 甲 + 乙
}

定 结果 = 求和(3, 5)
打印(结果)
```

### 斐波那契数列

```xinyu
函数 斐波那契(甲) {
  如果 甲 < 2 那么 {
    返回 甲
  }
  返回 斐波那契(甲 - 1) + 斐波那契(甲 - 2)
}

定 甲 = 0
当满足 甲 < 10 {
  打印("斐波那契(" + 甲 + ") = " + 斐波那契(甲))
  甲 = 甲 + 1
}
```

## 已知限制

- 不支持包管理器和 WASM 执行
- 不支持调试器（debug）功能
- 同时存在双字关键字和单字关键字两套，需注意区分
